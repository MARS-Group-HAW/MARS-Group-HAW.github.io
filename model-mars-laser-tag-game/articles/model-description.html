Model Description

The model includes the environment and agent interfaces.

The Environment

A 50x50 grid with barriers, hills, ditches, water, explosive barrels, and flag stands. Maps are square or rectangular.

Structures

  • Barrier: Blocks movement and vision.
  • Hill: Improves view but increases risk.
  • Ditch: Reduces visibility but adds cover.
  • Room: Section enclosed by barriers.
  • Water: Blocks movement but not sight.
  • ExplosiveBarrel: Damages nearby agents when hit.
  • FlagStand: Base and respawn point in Capture the Flag.

Items

  • Flag: Can be picked up by opponents, returned by teammates.

Game Mechanics

Vision

Dependent on stance, obstacles, distances, and visibility rules.

Movement

Based on D* Lite pathfinding. Stance affects delay:

  • Standing: 0 ticks
  • Kneeling: 2 ticks
  • Lying: 3 ticks

Tagging

Probability-based. Influenced by stance, terrain, and randomness.

Spots

Environmental features offer tactical advantages. Some occupiable, some not.
