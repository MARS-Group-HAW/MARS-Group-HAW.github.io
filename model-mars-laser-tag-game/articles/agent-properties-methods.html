Agent Properties and Methods

The IPlayerBody interface defines properties and methods that describe the agent and its capabilities. These are accessible by extending AbstractAgentMind.

Properties

General Properties

  • ActionPoints: Number of points available for actions each tick. Resets to 10 every tick.
  • Color: Team color.
  • Energy: Max 100. Decreases when tagged. <=0 means removal.
  • GamePoints: Score from tagging others. Bonus for knockouts.
  • CarryingFlag: Whether agent is carrying a flag.

Movement Properties

  • Position: Agent's (x, y) location.
  • Stance: Standing, Kneeling, or Lying. Affects visibility and movement.

Exploration Properties

  • VisualRange: Depends on stance. Standing: 10, Kneeling: 8, Lying: 5
  • VisibilityRange: Same scale as VisualRange

Tagging Properties

  • RemainingShots: Number of shots left.
  • WasTaggedLastTick: Was the agent tagged in last tick?
